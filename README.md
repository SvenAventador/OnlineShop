# Книжный мир — Интернет-магазин книг

> **ВНИМАНИЕ! Это тестовое приложение для выполнения практической работы по дисциплине «Поддержка и тестирование программных модулей».**

---

## О проекте

**«Книжный мир»** — это простой интернет-магазин по продаже книг, разработанный специально для отработки навыков ручного тестирования, анализа требований и оформления баг-репортов.

Приложение содержит **намеренно внесённые дефекты** (баги), которые студентам предстоит найти, задокументировать и оформить в виде Issues.

---

## Цель практической работы

1. Проанализировать требования к программному обеспечению.
2. Провести декомпозицию системы на функциональные блоки.
3. Разработать тестовую документацию:
   - Тест-план
   - Чек-лист
   - Тест-кейсы (минимум 12–15 штук)
4. Провести ручное тестирование методом «чёрного ящика».
5. Зафиксировать найденные дефекты в баг-трекере (GitHub Issues).
6. Применить знания о жизненном цикле дефекта и атрибутах баг-репорта.

---

## 🛠 Функциональные требования

### 1. Каталог товаров
- Отображение списка книг с полями:
  - Название
  - Автор
  - Жанр
  - Цена

### 2. Фильтрация
- Возможность фильтровать книги по жанрам:
  - Детектив
  - Фантастика
  - Роман
  - Научная литература

### 3. Поиск
- Поиск книг по названию или автору через поисковую строку.

### 4. Корзина
- Добавление книги в корзину (кнопка «В корзину» у каждой книги).
- Просмотр содержимого корзины (количество книг, общая стоимость).
- Удаление книги из корзины.
- Очистка корзины.

### 5. Оформление заказа
- Переход на страницу с формой заказа (ФИО, адрес, телефон).
